Blend Modes Help

The following table describes the available blend modes, their blending functions, and recommended uses.
Name Pixels are colored by... Recommended Use
Normal The depth of each layer is stored, and each pixel is colored by the layer closest to the camera. "Closer" layers completely block out "farther" layers. Normal blending also takes transarency into account. EM Data
Additive RGB values for each layer are first multiplied by that layer's alpha value, then summed. IF and LM (e.g. AT)
Subtractive RGB values for each layer are first multiplied by that layer's alpha value, then the sum of 1-(each color) is computed and used as the pixel value.
Multiply Colors obtained by multiplying RGB values across each layer.
